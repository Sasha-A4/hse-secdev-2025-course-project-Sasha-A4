# Evidence Directory

This directory contains security and compliance artifacts generated by CI/CD workflows.

## Structure

- `P09/` - SBOM and SCA artifacts from P09 workflow

## P09 Artifacts

Generated by `.github/workflows/ci-sbom-sca.yml`:

- `sbom.json` - Software Bill of Materials in CycloneDX JSON format
  - Generated using Syft (v0.120.0)
  - Contains all dependencies and their versions
  - Used as input for SCA scanning

- `sca_report.json` - Software Composition Analysis report
  - Generated using Grype (v0.78.0)
  - Contains detailed vulnerability information
  - JSON format for programmatic processing

- `sca_summary.md` - Human-readable summary of vulnerabilities
  - Aggregated counts by severity (Critical/High/Medium/Low/Negligible/Unknown)
  - List of Critical and High severity vulnerabilities
  - Next steps and recommendations

## Traceability

Each artifact includes:
- Commit SHA
- Workflow run ID
- Generation timestamp

This allows linking artifacts to specific code versions and CI runs.
